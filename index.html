IRC Anime Guide

Thanks to Hatsuyuki Fansubs

Are all your torrent links dead?

And you just wanna download anime through IRC/XDCC without all the bs?

STEP 1: Install an IRC Client

  1. Visit http://clicksandwhistles.com/download.php (only available for Windows)
  2. Click to download the program
  3. Open the ZIP file and extract the contents
  4. Run setup.exe
    NOTE: If you get a dialog asking you to download Visual C++ Runtime Libraries (x86), then press Install and follow the instructions
  5. In the installation guide, press Next
  6. Tick off I Agree, and press Next
  7. Click Next, and then Next again to start the installation

STEP 2: Set up the Connection

  1. Once installed, start the ClicksAndWhistles program
  2. Click on the Connect icon
  3. Enter a dummy nickname, name and email, and press OK
    NOTE: You have to type something in those fields, so just type nonsense
  4. Click on New Server... button
  5. In the Name field, type Rizon, and in the Address field, type irc.rizon.net. Press OK
    Only the address is important. You can name it whatever you want.
  6. Select Rizon,and press OK to connect
  7. Click on the Join icon
  8. Type in #nibl, and press OK
    NOTE: Joining the #nibl channel means you'll have access to all the servers and files on nibl.co.uk. However, if you find an unlisted server on another website, you might not be able to download the file without joining their channel.

STEP 3: Download anime files

  1. Go to this website for all the files: https://nibl.co.uk/bots.php
  2. In the search bar, type in the anime episode you want to download (preferably with the fansub name), and click Search

    Since the search bar on nibl.co.uk sucks, you may not always find your torrent even if it exists.

    Make sure of the following points:

    • Type the fansub group name at the beginning
    • Do not type in unnecessary info, just keep it to the anime name and episode number

  3. Click on the file you want to download (Click on its title under the Filename column)
    The Name column shows you the name of the server the file is on.
  4. Copy the text that appears in a popup
  5. Paste it in the text field on ClicksAndWhistles program, and hit Enter
  6. Click Yes in the popup that appears
  7. Choose where you want to download the file, and hit Save
    TIP: I like to keep a separate folder for my IRC/XDCC downloads
  8. Congratulations! You're now downloading the file
    NOTE: Depending on which file you're downloading, you most likely won't be able to download multiple files at once from the same server
